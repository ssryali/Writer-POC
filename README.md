# Xfinity Creative Companion — Nudge Strategizer

A Next.js application with 12 Writer AI agents that analyze and generate creative nudges for marketing briefs.

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm (comes with Node.js)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/ssryali/Writer-POC.git
cd Writer-POC
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Available Scripts

| Command | Description |
|---|---|
| `npm run dev` | Starts the development server at `http://localhost:3000` |
| `npm run build` | Builds the app for production |
| `npm run start` | Starts the production server (requires a build first) |
| `npm run lint` | Runs the linter |

## Usage

- **Generate all agents at once** — Fill in the inputs across the cards and click the **Generate Nudge** button at the bottom of the page.
- **Generate a single agent** — Open any card, enter your input, and click the **Generate** button inside that card.
